SIYANTRA AI SOLUTIONS – COMPANY WEBSITE

The official marketing website for Siyantra AI Solutions, an AI automation and consulting company focused on building intelligent systems for modern businesses, currently specializing in fleet and logistics automation.

Tagline:
Automate. Intelligent. Transform.

OVERVIEW

This website serves as the primary online presence for Siyantra AI Solutions. It showcases the company's services, business process, industry focus, and includes a contact form and an AI-powered chat assistant for lead generation and customer engagement.

LOCAL DEVELOPMENT

Requirements:

* Node.js version 18 or newer

Installation Steps:

1. Install dependencies
   npm install

2. Start development server
   npm run dev

3. Build production version
   npm run build

4. Preview production build
   npm run preview

Default Local URL:
http://localhost:5173

CHAT ASSISTANT CONFIGURATION

Current Mode:

* Local scripted assistant
* No backend required

Future AI Upgrade:

To integrate Claude AI:

1. Create a backend API endpoint
2. Store Claude API key securely on the server
3. Forward user messages to Claude
4. Return AI responses to the frontend
5. Enable AI mode inside ChatWidget.tsx

Security Note:
Never expose API keys in frontend code.
